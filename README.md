• Guia de Execução: 
1) Foi criado um blazor para rodar via local, e a pasta aberta no Code;
2) No code foi criada uma pasta "Models" e nela um arquivo tipo c# "Experiencia", onde foram criadas as propriedades usadas na página de Currículo;
3) Na pasta "Home", foram definidas as formatações e o texto que cada propriedade irá receber;
4) Por fim, no CMD, foi dado o comando dotnet run e, ao finalizar, foi gerado um caminho local que foi acessado num navegador para exibir a página já concluída.
• Tecnologia utilizada: Blazor WebAssembly
